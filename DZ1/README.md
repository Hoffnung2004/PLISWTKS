### Создание простого счётчика
Для дальнейшей демонстрации работы с ILA был создан простой счётчик с синхронным сбросом, кототорый считает до числа $8221319_{10}$, после чего обнуляется. 

```SystemVerilog
always_ff @(posedge w_clk_out1) begin
    if (rst) begin
        counter <= 28'd0;
    end else begin
    if (counter == 28'd8221319) begin
        counter <= 28'd0;
    end else begin
        counter <= counter + 28'd1;
    end
    end
end
```

### Сравнение VIO и ILA

| Характеристика           | VIO (Virtual Input/Output)    | ILA (Integrated Logic Analyzer) |
| :----------------------- | :---------------------------- | :------------------------------ |
| **Основное назначение**  | Управление и наблюдение       | Наблюдение                      |
| **Тип работы**           | Интерактивный                 | Пассивный                       |
| **Воздействие на схему** | Прямое вмешательство в работу | Только считывание               |
| **Глубина захвата**      | Текущее состояние             | Есть настраиваемый буфер        |
| **Аналогия**             | Пульт управления              | Осциллограф                     |
